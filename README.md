# ArduinoNano_Space_Downfall_code
A small Space Invaders-inspired game built using ChatGPT for an Arduino NanoESP32, a 0.96" oled display, a 3-pin potentiometer, and a limit switch. I'm calling this game Space Downfall since I don't wanna risk any copyright with Space Invaders lol.
## How to upload
Open the .ino file in ArduinoIDE and upload it to your board. Make sure you have the adafruit libraries to drive the oled display and other components. They're listed at the top of the code, and the drivers can be easily installed directly in ArduinoIDE in libraries.
